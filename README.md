# 🥊 LocuraUFC

LocuraUFC es una página web desarrollada como mi **primer proyecto personal**, centrada en el universo de la UFC. Creada hace más de dos años, esta web representa mis primeros pasos con **HTML, CSS y JavaScript**.

> ⚠️ Este proyecto **no ha sido actualizado recientemente** y refleja mis conocimientos y estilo de desarrollo en mis inicios. Lo mantengo como un recuerdo del aprendizaje y evolución que he seguido desde entonces.

---

## 📌 Características

- Información general sobre la UFC y sus luchadores
- Diseño básico con estructura en HTML
- Estilos aplicados con CSS puro
- Interactividad básica usando JavaScript

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🧠 Aprendizajes

Este proyecto me permitió:

- Comprender la estructura básica de una web
- Practicar la separación entre contenido, estilos y lógica
- Introducirme en la manipulación del DOM con JS
- Desarrollar pensamiento creativo en diseño web
- Manejo de eventos con JS

---

## 📌 Estado del proyecto

Este repositorio se conserva como **referencia histórica** de mi evolución como desarrollador. Aunque no está en desarrollo activo, es un símbolo importante de cómo comencé.
